Calcolo dimensioni di un file

In questa pagina potrai sapere quanto pesa un file con caratteristiche date

Questa sezione utilizza un programma progettato in C, riscritto in javascript, che prende in input dei parametri e restituisce la dimensione del file.
Il programma moltiplica i parametri e divide il risultato per 8 per ottenere la dimensione in byte e poi tramite un ciclo while ne calcola l'ordine di grandezza che può essere tra i seguenti:

Nome Simbolo Grandezza in byte Grandezza rispetto al multiplo precedente Ordine di grandezza
byte B 20B 20B 101
kilobyte kB 210B 210B 103
megabyte MB 220B 210kB 106
gigabyte GB 230B 210MB 109
terabyte TB 240B 210GB 1012
petabyte PB 250B 210TB 1015
exabyte EB 260B 210PB 1018
zettabyte ZB 270B 210EB 1021
yottabyte YB 280B 210ZB 1024
